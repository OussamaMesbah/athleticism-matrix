# Research: Durability Benchmarks (Injury Incidence)

Injury Incidence Rate (IIR) is generally measured per 1000 Athlete Exposures (AE).

| Sport                   | Game Injury Rate (per 1000 AE) | Risk Profile                    |
| :---------------------- | :----------------------------- | :------------------------------ |
| **Football (American)** | 35.9                           | Extreme (High Impact)           |
| **Rugby**               | 25.0 - 30.0                    | High (Sustained Impact)         |
| **Wrestling**           | 13.8 - 20.0                    | High (Musculoskeletal Trauma)   |
| **Soccer**              | 16.4 - 18.0                    | Moderate-High (Lower Extremity) |
| **MMA / Combat**        | 12.0 - 15.0                    | High (Traumatic Impact)         |
| **Basketball**          | 7.0 - 9.0                      | Moderate (Contact/Repetitive)   |
| **Baseball**            | 5.8                            | Moderate-Low                    |
| **Table Tennis**        | 0.5 - 1.0                      | Low (Repetitive Motion)         |
| **Golf**                | 0.1 - 0.5                      | Minimal (Overuse)               |

### Sources:
- Hootman et al. (2007)
- NCAA Injury Surveillance System (2015-2019)
- UFC PI Combat Sport Incidence Report
