# Study Summary: Haugen et al. (2018)
**Title**: New Records in Human Power
**Journal**: International Journal of Sports Physiology and Performance
**Date**: July 1, 2018

## Core Objective
To define the upper limits of aerobic and anaerobic power in elite athletes across various disciplines and provide reference values for scientists and practitioners.

## Key Data: VO2 Max (ml/kg/min)
| Category   | Athlete Type | Upper Limit (Relative) | Absolute L/min |
| :--------- | :----------- | :--------------------- | :------------- |
| **Male**   | XC Skiers    | ~90                    | ~7.5           |
| **Male**   | Cyclists     | ~90                    | -              |
| **Male**   | Runners      | ~90                    | -              |
| **Male**   | Rowers       | -                      | ~7.0           |
| **Female** | XC Skiers    | ~80                    | <5.0           |
| **Female** | Runners      | ~80                    | <5.0           |
| **Female** | Rowers       | -                      | <5.0           |

## Scientific Notes
- Maximal aerobic and anaerobic power are primary performance determinants in most sports.
- XC skiers consistently show the highest relative VO2 max values due to multi-limb involvement.
- Absolute power values (L/min) are highest in rowers due to larger body mass.
- Validity depends on calibrated apparatus and strict testing protocols.
