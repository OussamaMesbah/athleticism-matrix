# Research: Power Benchmarks (Vertical Jump)

Vertical jump is a primary metric for explosive power. Data primarily sourced from NBA/NFL combine records.

| Sport                 | Elite Benchmark (Inches) | Context                                               |
| :-------------------- | :----------------------- | :---------------------------------------------------- |
| **Basketball (NBA)**  | 35-42+                   | Elite "high flyers" (e.g., Wilt, Jordan, Zach LaVine) |
| **Football (NFL)**    | 33-46                    | Top scores usually DBs and WRs                        |
| **Volleyball**        | 32-40                    | High-performance outside hitters                      |
| **Soccer (Elite)**    | 24-30                    | High-level collegiate and pro defenders               |
| **Combat Sports**     | 22-28                    | Explosive shot-taking and power generation            |
| **Tennis**            | 15-22                    | Rapid vertical acceleration for serves                |
| **Endurance Running** | 10-15                    | Minimal explosive requirement                         |

### Historical Records:
- **Leonel Marshall** (Volleyball): ~50 inches
- **Michael Jordan** (NBA): ~46 inches (running)
- **Josh Imatorbhebhe** (NFL): 46.5 inches (2021)
